# WordPress Plugins

Plugins that add AEO schema, llms.txt generation, structured data, and AI crawler optimization to WordPress sites.

---

## Top Picks

**[AEO God Mode](https://aeogodmode.io/)**
The most complete AEO plugin for WordPress. Auto-detects Article/FAQ/HowTo/Product/LocalBusiness schema, generates llms.txt, manages AI crawlers, and includes a citation tracker that queries four AI engines weekly. Free core on WordPress.org. Pro adds analytics. Agency tier for unlimited client sites.

**[AEO Engine](https://wordpress.org/plugins/alquingadev-aeo-schema/)**
Generates 9 JSON-LD schema types, auto-detects FAQ pairs and HowTo steps, and produces `/llms.txt` and `/llms-full.txt` files. Smart conflict detection with Yoast and RankMath — no manual deduplication needed.

**[Rank Math SEO](https://rankmath.com/)**
Feature-rich SEO suite with schema generator, FAQ block, HowTo block, and rich snippets that directly feed structured data to AI crawlers. One of the most complete free options available.

## Notes on llms.txt

llms.txt is a proposed standard (like robots.txt) for communicating site content to LLM crawlers. Already implemented by Stripe, Zapier, Cloudflare, and Anthropic. Supported by OpenAI and Perplexity. Several plugins above generate it automatically — worth enabling if you're on any of them.
